# /public/sounds/ — Market Wars Display audio

วางไฟล์เสียงที่นี่ **ตั้งชื่อให้ตรงเป๊ะ** (= key ใน `lib/sound.ts`)
ฟอร์แมต `.mp3` 44.1kHz ~128–192kbps · โค้ด graceful: ไฟล์ไหนยังไม่มา = เงียบ ไม่พัง

## BGM (loop, 60–120s, ต้อง author แบบ loop เนียน)
- [ ] `bgm_lobby.mp3` — lobby (ต้อนรับ ชิล สดใส)
- [ ] `bgm_play.mp3` — research / invest / chance_card (เกมโชว์เบาๆ)
- [ ] `bgm_suspense.mp3` — year_intro / market_open / event (ลุ้น ตึง)
- [ ] `bgm_results.mp3` — event_result / golden_deal / results / leaderboard (คึก พลังงานสูง)
- [ ] `bgm_final.mp3` — final (ฉลอง ชัยชนะ)

## SFX (one-shot)

- [ ] `sfx_market_bell.mp3` — market_open (กระดิ่งตลาด, 1–2s) ← ใช้แล้วใน B16a
- [ ] `sfx_countdown.mp3` — 10 วิสุดท้าย (beep, <0.3s) ← ใช้แล้วใน B16a
- [ ] `sfx_timeup.mp3` — หมดเวลา (chime, 0.5–1s) ← ใช้แล้วใน B16a
- [ ] `sfx_join.mp3` — ผู้เล่นเข้า (ป๊อป, 0.3–0.7s) — รอ B16b
- [ ] `sfx_card_flip.mp3` — เปิดการ์ด (0.5–1s) — รอ B16b

- [ ] `sfx_cash.mp3` — ได้เงิน (ka-ching, 0.5–1.5s) — รอ B16b/d

- [ ] `sfx_rankup.mp3` — อันดับขึ้น (whoosh-up, 0.5–1.5s) — รอ B16c
