# YoungGen Portfolio Challenge — สรุปสิ่งที่แก้ไข (V3 → V6)

**สำหรับทีมงานทดสอบ** · อัปเดต 03 ก.ค. 2026
เอกสารนี้อธิบาย "อะไรเปลี่ยนไปบ้าง" แบบเข้าใจง่าย + "ทดสอบยังไง" เพื่อให้ทีมช่วยเช็คก่อนงานจริง (6–10 ก.ค.)

> V3–V6 = เสร็จแล้ว (พร้อมทดสอบ) · **V6 = งานสุดท้าย** — จอทั้งหมดพร้อม 4–8 ทีม + EN ครบ + rejoin ต้นวัน + สลับกลุ่ม

## ✅ V3 — เสร็จแล้ว พร้อมทดสอบ

### 1. ปรับน้ำหนักลงทุนทีละ 5% (เดิม 10%)
- **เปลี่ยนอะไร:** ปุ่ม +/- ในหน้าจัดพอร์ตของทีม ขยับทีละ 5% แทน 10% → จัดพอร์ตได้ละเอียดขึ้น
- **ทดสอบ:** เข้าหน้าทีม → กด + ที่ asset ใดก็ได้ → ต้องขึ้นทีละ 5 (5, 10, 15, …) · รวมให้ครบ 100% แล้วกด Submit ได้ปกติ

### 2. แก้บั๊ก Submit ไม่ผ่านตอนใช้ 5%
- **เปลี่ยนอะไร:** ก่อนหน้านี้ถ้าลง 5/15/25% แล้วกด Submit จะเด้ง error "must be a multiple of 10" → แก้แล้ว
- **ทดสอบ:** ลงน้ำหนักที่ลงท้าย 5 (เช่น 15% + 35% + 50%) แล้วกด Submit → ต้องผ่าน ไม่มี error

### 3. ยกเลิกเพดาน Crypto 20%
- **เปลี่ยนอะไร:** เดิม Crypto ใส่ได้สูงสุด 20% · ตอนนี้ใส่ได้ถึง 100% (ไม่มีเพดาน) · ป้าย "max 20%" หายไป
- **ทดสอบ:** Challenge 6 หรือ 7 → ลอง + Crypto ขึ้นไปเรื่อยๆ → ต้องขึ้นได้เกิน 20% จนถึง 100% · ไม่มีป้าย max

### 4. Submit แล้วแก้ไม่ได้ (ล็อกเลย)
- **เปลี่ยนอะไร:** เอาปุ่ม "✏️ EDIT ALLOCATION" ออก · พอกด Submit แล้วพอร์ตล็อกทันที มีป้าย "🔒 Allocation locked" แทน
- **ทดสอบ:** กด Submit → ต้องไม่มีปุ่มแก้ · เห็นสรุปพอร์ตของทีมตัวเอง + ป้ายล็อก · รอ MC ไปต่อ

### 5. จอใหญ่ (Projector) ไม่โชว์สินทรัพย์ที่ยังไม่ปลดล็อก
- **เปลี่ยนอะไร:** ตอนเฉลยผลตลาด เดิมโชว์ Oil/Crypto เป็น 0% ในรอบที่ยังเล่นไม่ได้ → ตอนนี้ซ่อน โชว์เฉพาะที่ปลดล็อกจริง
- **ทดสอบ:** ดูจอใหญ่ตอนเฉลยผลแต่ละ Challenge
  - Ch1–4: เห็น 6 สินทรัพย์
  - Ch5: เห็น 7 (มี Oil, ไม่มี Crypto)
  - Ch6: เห็น 7 (มี Crypto, **ไม่มี Oil** แล้ว)
  - Ch7: เห็นครบ 8

### 6. ภาษาอังกฤษบนจอที่แก้ไปแล้ว
- **เปลี่ยนอะไร:** จอเฉลยผลตลาด (Event) เป็นอังกฤษล้วนแล้ว (จอที่เหลือจะตามใน V6)
- **ทดสอบ:** ดูจอ Event → ไม่มีภาษาไทยปน (ยกเว้นสัญลักษณ์เงินบาท ฿ ซึ่งตั้งใจเก็บไว้)

## ✅ V4 — เสร็จแล้ว พร้อมทดสอบ (เอา timer ออก + เปิดผลพร้อมกัน)

### 7. เล่นแบบไม่จับเวลา
- **เปลี่ยนอะไร:** เอานาฬิกานับถอยหลังออกทุกหน้า + เสียงนับถอยหลังหายด้วย → MC กดไปต่อเอง ไม่มีเวลาบีบ
- **ทดสอบ:** เล่นทุก phase (จัดพอร์ต/ตลาด/ผล) → ต้องไม่มีนาฬิกาบนจอไหนเลย + ไม่มีเสียงติ๊กนับถอยหลัง

### 8. Submit แล้วรอเปิดผลพร้อมกันทุกทีม
- **เปลี่ยนอะไร:** พอทีม Submit จอใหญ่ขึ้นแค่ "ทีมไหน Submit แล้ว" (✓ ไม่โชว์ %) → MC กด **"🔓 Reveal Allocations"** → จอใหญ่โชว์พอร์ตทุกทีมพร้อมกัน + คำอธิบายสีสินทรัพย์ → MC กด "Next → Market Open" เล่นต่อ
- **ทดสอบ:**
  1. เข้าจัดพอร์ต → ให้ 2-3 ทีม Submit → จอใหญ่ขึ้นแค่ชื่อ + ✓ (ยังไม่เห็น %)
  2. MC กด "🔓 Reveal Allocations" → จอใหญ่โชว์แถบพอร์ตของทุกทีมพร้อมกัน + คำอธิบายสี
  3. บนมือถือทีมตอนนี้เห็นข้อความ "Allocations are on the big screen — watch together" (ไม่เห็นของทีมอื่น)
  4. MC กด "Next → Market Open" → เล่นต่อปกติ

## ✅ V5 — เสร็จแล้ว พร้อมทดสอบ (ปรับหน้าจบเกม)

### 9. หน้าประกาศผลจบเกมกระชับขึ้น (ตัดหน้า "Step Reward")
- **เปลี่ยนอะไร:** หน้าจบเกมเหลือ 2 สเต็ป → ① ประกาศแชมป์ (ลุ้น #3→#2→#1) → ② ตารางอันดับสุดท้าย · หน้ารางวัลพิเศษ (Special Awards) ถูกตัดออก
- **ทดสอบ:** เล่นจบ Challenge 7 → MC กด **เฉลยแชมป์** → จอใหญ่ไล่ Top 3 (มี confetti) → กด **② Ranking** → ต้องไม่มีหน้ารางวัลคั่นระหว่างกลาง · ปุ่ม Step ของ MC ต้องมีแค่ ① Podium / ② Ranking

### 10. จอมือถือกันสปอยล์ตอนลุ้นแชมป์
- **เปลี่ยนอะไร:** ตอน MC ประกาศแชมป์ (①) มือถือทีมจะยังไม่โชว์อันดับ/ผลตัวเอง — ขึ้นหน้า "👀 Watch the big screen" แทน · เห็นผลตัวเองตอนเปิด ② Ranking เท่านั้น
- **ทดสอบ:** ตอนอยู่หน้า Podium → เปิดมือถือทีม → ต้องเห็นแค่ "Watch the big screen / result hidden" (ไม่เห็นอันดับ) → พอ MC กด ② Ranking → มือถือถึงโชว์ #อันดับ + กำไร + Top 5

### 11. ตารางอันดับสุดท้าย + หน้าจบ = สะอาดขึ้น
- **เปลี่ยนอะไร:**
  - เอา **เส้นเปรียบเทียบ (benchmark กรอบฟ้า)** ออกหมด → โชว์เฉพาะทีมที่เล่นจริง (คงป้าย 🎯 ทุ่มกระจุก / 🧺 กระจาย ไว้)
  - เอา **รางวัล "นักลงทุนกระจายความเสี่ยง" (🏅)** ออกจากทุกจอหน้าจบ (ตาราง/แท่นแชมป์/มือถือ) — สอดคล้องกับการตัดหน้า Special Awards
  - มือถือ: เอาการ์ด **"YOUR STATS" (Quiz ถูก / Chance Card)** ออก (YG ไม่มี quiz/การ์ดดวง — เดิมขึ้น 0/14 กับ ฿0 รกๆ)
- **ทดสอบ:**
  1. หน้า ② Ranking (จอใหญ่) → ต้องไม่มีการ์ดกรอบฟ้าประ (bm_best/ฝากเงิน/กระจายเท่ากัน/bm_worst) · ไม่มี 🏅 · ไม่มีคำอธิบายกรอบฟ้าใต้ตาราง — เห็นแต่การ์ดทีมจริง
  2. หน้าแท่นแชมป์ (Podium) → ต้องไม่มีป้าย "🏆 ชนะ 2 รางวัล" ใต้ชื่อผู้ชนะ
  3. มือถือหน้าจบ (② Ranking) → เห็น #อันดับ → กำไร → กราฟผลตอบแทนรายปี → Top 5 → Thank you · **ต้องไม่มี** การ์ด YOUR STATS / Quiz / Chance Card / รางวัล

## ✅ V6 — เสร็จแล้ว พร้อมทดสอบ (จอ 4–8 ทีม + ภาษาอังกฤษครบ + rejoin + สลับกลุ่ม) — งานสุดท้าย

### 12. จอใหญ่ออกแบบใหม่สำหรับ 4–8 ทีม (การ์ดใหญ่ อ่านออกบนโปรเจกเตอร์)
- **เปลี่ยนอะไร:** เดิมจอ inherit layout ~70 คนจาก Kids Camp (การ์ดเล็ก/แบ่งหน้า) → จัดใหม่เป็นแถวใหญ่ 4 จอ:
  - **Leaderboard (อันดับ):** แถวใหญ่ + แถบเงิน (ยาวตามสัดส่วนผู้นำ) + 🥇🥈🥉 + ลูกศรขึ้น/ลง + 🐎 ม้ามืด
  - **Reveal (เปิดพอร์ต):** แถวใหญ่ + แถบสัดส่วนสินทรัพย์ทุกทีมพร้อมกัน + ป้าย 🎯 กระจุก / 🧺 กระจาย + คำอธิบายสี
  - **Results (ผลรอบ):** แท่งซ้าย-ขวาจากเส้นกลาง (เขียวขวา=กำไร / แดงซ้าย=ขาดทุน) เรียงตามผลรอบ
  - **Final Standings (อันดับจบเกม):** แถวใหญ่ + แถบเงิน + 🥇🥈🥉 + ป้ายกลยุทธ์ + %ผลตอบแทน + เงินสุดท้าย
- **ทดสอบ:** เปิดจอใหญ่ ดูทุก phase — การ์ดต้องใหญ่ อ่านออกจากหลังห้อง · ตัวเลขเงินต้องไม่ตกขอบจอ · ทุกแถวอยู่ครบไม่ล้น

### 13. ภาษาอังกฤษครบทุกจอ (น้องๆ)
- **เปลี่ยนอะไร:** กวาดภาษาไทยที่เหลือทุกจอ Display + จอมือถือทีมเป็นอังกฤษ (จอ MC พิธีกรเก็บไทยไว้ให้อ่านง่าย · สัญลักษณ์ ฿ เก็บไว้) · "ปี/Year" → "Challenge" ทุกจุด · หน้าแนะนำ challenge บนมือถือแก้ list ให้ตรงเกม (Allocate → Reveal → Market → Results) + เงินที่โชว์ = ยอดจริงของทีม (ไม่ใช่ 1,000,000 ตายตัว)
- **ทดสอบ:** ไล่ทุกจอ ไม่มีภาษาไทยปน (ยกเว้น ฿ + จอ MC) · เห็นคำว่า "Challenge N" ไม่ใช่ "ปีที่ N"

### 14. QR ให้ทีม rejoin ง่ายๆ ต้นวัน (เล่นหลายวัน)
- **เปลี่ยนอะไร:** หน้า Challenge Brief (ตอนเริ่มแต่ละ challenge) มี **QR มุมขวาบน** + room code + bit.ly/portchallenge → ต้นวันถัดไปทีมสแกน + พิมพ์ชื่อทีมเดิม = กลับเข้าเกมพร้อมพอร์ต/เงินเดิม
- **สำคัญ:** **ห้ามกด "จบเกม" ระหว่างวัน** — จบวันแค่ปิดโน้ตบุ๊ก (ห้องยังเปิดอยู่ ทีม rejoin ได้) · ถ้ากดจบเกม = ทีมเข้าไม่ได้อีก
- **ทดสอบ:** ต้นแต่ละ challenge เห็น QR มุมขวาบน · ลองปิดจอทีม → เปิดใหม่ → สแกน QR → พิมพ์ชื่อเดิม → ต้องกลับเข้าพร้อมเงิน/พอร์ตเดิม

### 15. ปุ่มสลับกลุ่มใหม่ (End & New Room) ที่จอ MC ตอนจบ
- **เปลี่ยนอะไร:** จบเกมกลุ่มหนึ่งแล้ว จอ MC มีปุ่ม **"🔄 จบเกม + สร้างห้องใหม่"** → สร้างห้องใหม่ให้เลย ไม่ต้องถอยออกไปหน้าแรก · ใช้ตอนรับกลุ่มถัดไป (มี confirm กันกดพลาด)
- **ทดสอบ:** หน้าจบเกม → กดปุ่มนี้ → ยืนยัน → ต้องเข้าห้องใหม่ (room code ใหม่) พร้อมรับทีมใหม่

### 16. แก้บั๊กจอใหญ่ล้น/ตกขอบบนโปรเจกเตอร์ 1080p
- **เปลี่ยนอะไร:** จอ final (ลุ้นแชมป์ / podium / อันดับจบ) เคยล้น/เลื่อนตกขอบบนจอที่สูงกว่า 720px (เช่นโปรเจกเตอร์ 1080p) → แก้ให้ fit พอดีทุกขนาดจอแล้ว
- **ทดสอบ:** เปิดจอใหญ่บน**โปรเจกเตอร์/หน้าจอ 1080p จริง** (ไม่ใช่แค่ในหน้าต่างเล็ก) แล้วดูหน้าจบเกม — ทุกอย่างต้องอยู่ในจอ ไม่ตกขอบล่าง

## หมายเหตุการทดสอบ (V6 — สำคัญสุดก่อนงาน 6 ก.ค.)
- **เปิดจอใหญ่บนโปรเจกเตอร์/หน้าจอ 1080p จริง** (ไม่ใช่หน้าต่าง browser เล็ก) — เล่นครบ 7 Challenge อย่างน้อย 1 รอบเต็ม
- เช็คทุกจอว่า **การ์ดใหญ่พอ อ่านจากหลังห้องได้** + **ตัวเลขเงินไม่ตกขอบ** + **หน้าจบไม่ล้น**
- ทดสอบ **rejoin ต้นวัน:** ปิดจอทีม → สแกน QR หน้า Challenge Brief → พิมพ์ชื่อเดิม → กลับเข้าพร้อมเงิน/พอร์ตเดิม
- ทดสอบ **End & New Room:** จบเกม → กดปุ่มสลับกลุ่ม → เข้าห้องใหม่
- เจอปัญหาแจ้งกลับได้เลย

## หมายเหตุการทดสอบ V3 (เดิม)
- ทดสอบบนลิงก์ deploy จริง (Vercel) — เปิด 3 จอ: ทีม (มือถือ) / MC (โน้ตบุ๊ก) / จอใหญ่ (โปรเจกเตอร์)
- ลองเล่นครบทั้ง 7 Challenge อย่างน้อย 1 รอบเต็ม (dry-run)
- เจอปัญหาแจ้งกลับได้เลย — โดยเฉพาะเรื่องน้ำหนัก 5% / Crypto เกิน 20% / จอใหญ่ตอนเฉลยผล
